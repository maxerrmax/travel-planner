# ✈️ AI Travel Planner

A Flask web application that generates personalized travel itineraries using OpenAI.

## 🚀 Features
- AI-generated travel plans
- Custom destination, duration and preferences
- Simple web interface (Flask + HTML)

## 🧠 Tech Stack
- Python
- Flask
- OpenAI API
- HTML/CSS


## ⚙️ Setup

```bash
git clone https://github.com/your-username/travel-planner
cd travel-planner
pip install -r requirements.txt
python run.py
