# 🌍 AI Travel Planner: Your Intelligent Itinerary Creator

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![AI-Powered](https://img.shields.io/badge/AI-GPT--4%20/%20Gemini-blueviolet)

**AI Travel Planner** is a modern web application designed to revolutionize how we plan trips. By leveraging cutting-edge Artificial Intelligence, the app generates fully personalized, day-by-day itineraries based on user preferences, budget, and travel dates.

---

## ✨ Key Features

- 🤖 **AI-Driven Itineraries:** Generate detailed schedules (morning, afternoon, evening) in seconds.
- 📍 **Global Destination Support:** Plan trips for any city or hidden gem worldwide.
- 💰 **Budget Management:** Filter activities based on "Economy," "Balanced," or "Luxury" tiers.
- 🏨 **Smart Accommodations:** Get hotel suggestions strategically located near your main interests.
- 🍴 **Curated Food Spots:** Restaurant recommendations tailored to your dietary preferences and local specialties.
- 🗺️ **Interactive Maps:** Visualise your journey with integrated map pins (Google Maps/Mapbox).

---

## 🚀 Tech Stack

The project is built with the following technologies:

- **Frontend:** [React.js](https://reactjs.org/) / [Next.js](https://nextjs.org/) with [Tailwind CSS](https://tailwindcss.com/).
- **Backend:** [Node.js](https://nodejs.org/) (Express) or [Python](https://www.python.org/) (FastAPI).
- **AI Integration:** OpenAI API (GPT-4), Google Gemini, or Anthropic Claude.
- **Database:** [Supabase](https://supabase.com/) / [Firebase](https://firebase.google.com/) / [MongoDB](https://www.mongodb.com/).
- **Maps API:** Google Maps Platform / Mapbox.
